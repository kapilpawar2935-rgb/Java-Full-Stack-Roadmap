# Day 10 – Methods / Functions (Java)

## Topics Covered
- Introduction to Methods
- Method syntax and structure
- Methods with parameters
- Methods with return values
- Calling methods from main()
- Practical method-based problem solving

## Programs
- SumMethod.java – Sum of two numbers using method
- MaxTwoNumbers.java – Find maximum of two numbers using method
- EvenOddMethod.java – Check even or odd using method
- FactorialMethod.java – Factorial calculation using method
- SquareMethod.java – Square of a number using method
- FactorialUsingMethod.java - Find factorial 

## Learning Outcome
Learned how to create reusable functions (methods), pass parameters, return values, and write cleaner and modular Java programs. These concepts are essential for problem solving, interviews, and real-world application development.
